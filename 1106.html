소개 (About Me)

안녕하세요! 사용자 경험(UX)과 아름다운 디자인을 코드로 구현하는 것을 좋아하는 프론트엔드 개발자 홍길동입니다. 저는 기술적인 도전을 즐기며, 특히 반응형 웹 디자인과 인터랙티브한 UI 구현에 큰 관심을 가지고 있습니다. 이 포트폴리오는 2025년 11월 6일자 UI 디자인 시스템 명세서를 기반으로 제작되었습니다.

'Sunset Orange' 컬러 팔레트와 'Modern Sans' 폰트 시스템을 적용하여 가독성과 시각적 즐거움을 동시에 추구했습니다.

프로젝트 (Projects)

Project Sunset

'Sunset Orange' 팔레트를 메인으로 사용한 E-commerce 웹사이트입니다. CSS Grid와 Flexbox를 활용하여 유연한 반응형 레이아웃을 구현했습니다.

Modern Grid Blog

'CSS Grid Modern' 시스템을 적용하여 제작한 개인 블로그입니다. 'auto-fit'과 'minmax'를 활용하여 콘텐츠 중심의 유연한 레이아웃을 완성했습니다.

Soft Components UI Kit

'Rounded Soft' 스타일 가이드를 기반으로 제작한 React 컴포넌트 라이브러리입니다. 12px 모서리 둥글기와 부드러운 그림자 효과를 적용했습니다.
